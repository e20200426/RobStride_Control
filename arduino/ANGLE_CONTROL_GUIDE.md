# 🎯 0-360度角度控制快速指南

## 🚀 立即开始使用角度控制

### 📖 角度输入系统

现在 `simple_joint_control` 项目支持 **0-360度角度输入**，让控制更加直观！

### 🎪 角度对应关系
```
角度     位置说明
0°     = 起始位置/正前方
45°    = 东北方向
90°    = 正右方
135°   = 东南方向
180°   = 正后方
225°   = 西南方向
270°   = 正左方
315°   = 西北方向
360°   = 回到起始位置(0°)
```

## 📱 快速命令测试

### 基础角度测试
```bash
# 打开串口监视器，依次输入：
0        # 回到起始位置
90       # 转到90度（正右方）
180      # 转到180度（正后方）
270      # 转到270度（正左方）
0        # 回到0度
stop     # 停止
```

### 精确角度测试
```bash
22.5     # 22.5度
67.5     # 67.5度

0        # 回到起点
```

### 超大角度测试
```bash
450      # 450度 → 自动转换为90度
720      # 720度 → 自动转换为0度
-90      # -90度 → 按弧度处理
```

## 🧠 智能识别规则

程序会智能判断输入类型：

| 输入范围 | 识别结果 | 示例 |
|----------|----------|------|
| **0-360** | 角度 ✅ | `90` → 90度 |
| **小弧度值** | 弧度 | `1.57` → π/2弧度 |
| **>360度** | 归一化角度 | `450` → 90度 |
| **负数** | 弧度 | `-1.57` → -π/2弧度 |

## 📊 实时反馈格式

程序会显示两种角度格式：
```
位置: 1.571 rad (90.0° -> 0-360°显示: 90.0°), 速度: 0.15 rad/s
         │      │                 └─ 0-360度显示角度（直观）
         │      └─ 实际计算角度
         └─ 弧度值（内部计算）
```

## 🔧 使用步骤

### 1. 上传程序
```bash
1. Arduino IDE打开: simple_joint_control.ino
2. 选择开发板: ESP32 Dev Module
3. 上传程序
```

### 2. 开始控制
```bash
1. 打开串口监视器（115200）
2. 等待"就绪！支持0-360度角度输入"提示
3. 直接输入角度数值开始控制
```

### 3. 角度控制示例


## 🔍 故障排除

### 问题：角度不准确
```bash
解决方案：
1. 重新设零：输入"zero"
2. 检查机械零点是否正确
3. 测试0度和180度是否准确
```

### 问题：电机振动
```bash
这个程序已经优化防振动
如果仍有振动：
1. 检查机械安装
2. 确认电源稳定
3. 输入"stop"停止测试
```

### 问题：角度识别错误
```bash
如果输入弧度被误识别为角度：
1. 使用小数弧度值：1.57, 3.14
2. 明确输入：1.5708 表示π/2
3. 查看串口输出的识别结果
```
