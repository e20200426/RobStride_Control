# ⚡ 快速开始指南

## 🚀 5分钟快速上手

### 1️⃣ 硬件连接
```
ESP32      CAN模块    小米电机
GPIO5  ←→ TX   ←→ CAN_H
GPIO4  ←→ RX   ←→ CAN_L
3.3V   ←→ VCC  ←→ 12-24V
GND    ←→ GND  ←→ GND
```

### 2️⃣ Arduino IDE设置
1. 安装ESP32开发板支持
2. 选择开发板：ESP32 Dev Module
3. 安装库：ESP32-TWAI-CAN

### 3️⃣ 上传程序
1. 打开 `mi_motor_control.ino`
2. 选择正确的串口
3. 点击上传

### 4️⃣ 运行测试
1. 打开串口监视器（115200波特率）
2. 观察初始化过程
3. 电机将自动开始转动测试

## 📱 串口命令测试

输入以下命令测试电机：
```
stop          # 停止
enable        # 使能
speed 2.0     # 设置速度2 rad/s
pos 1.57      # 转到90度位置
help          # 查看帮助
```

## 🔧 如果出现问题

1. **无法上传**：检查ESP32驱动和串口连接
2. **电机无响应**：检查CAN连接和电源
3. **编译错误**：确认ESP32开发板和库已安装
